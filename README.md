# Interactive Calculator

Kalkulator interaktif dengan history tanpa batas, fungsi memori, dan dukungan keyboard. Responsive untuk desktop & mobile.

---

## Fitur
- Operasi aritmatika: + − × ÷  
- Titik desimal dan persen (%)  
- Perhitungan berantai (menggunakan prioritas operator dasar)  
- Penanganan error (mis. pembagian dengan nol menampilkan "Error")  
- Keyboard support: 0–9, ., +, -, *, /, Enter(=), Backspace(CE), Esc(C)  
- History tanpa batas
- Memory: M+, M-, MR, MC
